                                                          DESCRIPTION
This is a program that asks you 60 questions in a randomized order and does scoring logic on them to tell you what your top 3 archetypes are (which triangulate your true archetype),
as well as your introversion/extroversion balance, which animal represents your introverted and extroverted selves, your mythical creature, and your alchemical elemental alignment from greatest to least.

                                   FOLLOW THE GUIDELINES OF THIS SECTION BEFORE TAKING THE TEST
This program has 2 modes: test phase and person phase. You need to run it in test phase before running the person phase just so you see which directory the resulting folder gets saved to,
called your "working directory." After that, run the program again and select option "2" for "person phase." It will ask your name, which you enter in before answering the 60 prompted questions.
The resulting folder containing the four files that present your results will be named after the name you give the program. READ THE NEXT SECTIONS BEFORE LAUNCHING IT.

                                                   OPERATIONAL PREREQUISITES
This code uses the programming language "Python." This section is for you if you're new to computing programming. Download "Visual Studio Code" from 
this link if you don't already have a code editor -> https://code.visualstudio.com/download
Once you've downloaded it, install Python version 3+ from this link -> https://www.python.org/downloads/

Then you'll need to open the program "Windows Powershell", running it as an administrator, and paste the line of code that's on the next line before running the program:

pip install numpy matplotlib pandas

hit enter

Now you just need to download the Animalquiz2.py script. Right click on it once it's downloaded, select "open with" and choose Visual Studio Code.

IF YOU DON'T KNOW HOW TO RUN CODE: 
All you do is save the script by pressing "file" then "save" from the drop-down menu on the top left, then press the play button in the top right corner. The rest of the quiz should be pretty intuitive.
